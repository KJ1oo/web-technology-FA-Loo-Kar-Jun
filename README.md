# Campus Lost & Found Management System

## Project Description
This project is a web-based system that allows campus users to report lost or found items and view existing reports. Administrators can manage submissions through an admin dashboard.

## Features
- View lost and found items
- Submit new item reports
- Admin login and dashboard
- Manage item records

## Technologies Used
- Node.js
- Express.js
- SQLite
- HTML / CSS / JavaScript

## Installation
1. Clone the repository
2. Install dependencies

npm install

3. Start the server

node app.js

4. Open in browser

http://localhost:3000
